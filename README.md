# 🛰️ AssiWorks - Daejeon — 대전 도시 상세 위성 변화 모니터링

AssiWorks - Daejeon 에이전트가 대전광역시의 주요 **도심·연구단지·산업단지·택지·하천·호소** 구역을 순환하며, 가장 최신 Sentinel-2(ESA, 10m) 위성영상에서 도시의 **건설·도시개발·토목·녹지·하천·지표교란 변화**를 분석하고, 같은 구역의 과거 같은 계절과 비교 검증해 **의미 있는 변화 신호일 때만** 발행하는 리포트입니다. 하루 여러 건을 **날짜/시각 슬롯**으로 발행합니다.

> ⚠️ **추정치·공개정보 안내**: 본 콘텐츠는 공개된 Sentinel-2(ESA Copernicus) 위성영상을 AI·알고리즘이 자동 분석한 **추정 결과**로, 사실과 다를 수 있습니다. 대상 좌표는 공개 지도 기반 근사 중심점이며 정밀 측지값이 아닙니다. 본 자료는 대전 도시 변화를 폭넓게 관찰하기 위한 참고용이며, 행정·법적 판단이나 특정 개인·사유지에 대한 감시 목적이 아닙니다. 정밀 측량·현장조사를 대체하지 않습니다.
